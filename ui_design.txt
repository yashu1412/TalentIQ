2. 3D Module Visual Reference
All 6 modules are built with React Three Fiber + @react-three/drei + @react-three/rapier. Each preview below shows the rendered visual alongside its technical specification.
2.1 HeroGlobe — Landing Page 3D Scene
Fig 1 — HeroGlobe3D: rotating Earth with talent connection arcs and star particles
Component
components/3d/HeroGlobe.tsx
Geometry
SphereGeometry(1.4, 64, 64) with custom GLSL shader — dark ocean, teal continents
Arcs
TubeGeometry between random lat/lng pairs; dashOffset animation for flow effect
Particles
Points (BufferGeometry) — 2000 star particles on sphere surface
Lighting
AmbientLight(0.3) + PointLight teal front + PointLight navy rear
Camera
PerspectiveCamera, fov=60, position=[0, 0, 3.5]
Animation
useFrame: Y-axis rotation at 0.003 rad/frame; arcs fade in with lerp opacity
Interaction
Pointer events: hover slows rotation; scroll wheel adjusts camera Z depth
Performance
instancedMesh for arcs; DPR capped at 1.5; disposed on unmount
Fallback
CSS gradient sphere with CSS animation if WebGL unavailable
2.2 ResumeOrb — Resume Processing Visualization
Fig 2 — ResumeOrb3D: state-reactive icosahedron with ring progress and physics particles
Component
components/3d/ResumeOrb.tsx
Geometry
IcosahedronGeometry(1.2, 4) — subdivided facets for organic look
Material
MeshStandardMaterial with emissive teal glow; roughness=0.3, metalness=0.7
States
idle: slow pulse  |  processing: noise displacement + fast spin  |  done: burst
Vertex Shader
simplex noise displacement; amplitude float tied to parse progress value
Progress Ring
TorusGeometry around orb; dashOffset animates from 2π → 0 as parsing completes
Particles
On done: 80 particles via Rapier RigidBody physics, gravity-fall off screen
Labels
@react-three/drei <Text> floating above: 'Extracting Skills…' / 'Done!'
Canvas
400×400px, embedded in upload card, transparent background
Props
status: 'idle' | 'processing' | 'done' | 'error'  +  progress: 0–100
2.3 SkillGraph3D — Job Match Visualization
Fig 3 — SkillGraph3D: interactive physics node graph — matched (teal), missing (rose), bonus (amber)
Component
components/3d/SkillGraph3D.tsx
Node Types
Matched = teal sphere  |  Missing = rose sphere  |  Bonus = amber sphere
Node Size
Proportional to skill importance score; range 0.08–0.25 unit radius
Physics
@react-three/rapier RigidBody: nodes float with spring force, repel each other
Edges
Line geometry between related skills; opacity = relationship strength score
Labels
drei <Billboard> + <Text> — always face camera; fade when overlapping
Camera
OrbitControls; auto-rotate 0.5 rpm; damping=0.1
Interaction
Click node: tooltip with skill name + learning resource link
Center Orb
Large glowing sphere showing match % via radial GLSL gradient shader
Props
matchData: { matched[], missing[], bonus[], matchScore: number }
2.4 InterviewAvatar3D — AI Interviewer
Fig 4 — InterviewAvatar3D: AI interviewer with morph-target mouth and eye tracking
Component
components/3d/InterviewAvatar3D.tsx
Head
IcosahedronGeometry with violet emissive material; subdivisions morph for 'thinking'
Eyes
SphereGeometry pupils; pupil raycasts toward mouse pointer each frame
Mouth
MorphTargetInfluences synced to TTS audio amplitude via Web Audio AnalyserNode
States
idle: head bob  |  thinking: slow rotation + glow  |  speaking: morph + burst
Audio Sync
useRef AudioContext analyser; useFrame reads frequency data → drives morph weight
Parallax
Camera lerp toward mouse position (±5°) gives depth illusion on mouse move
Scan Lines
Horizontal scan-line overlay texture for sci-fi AI aesthetic
Speaking Ring
Pulsing concentric ring animations around head when speaking state is active
Props
state: 'idle' | 'thinking' | 'speaking'  +  audioStream?: MediaStream
2.5 ScoreRing3D — Interview Score Display
Fig 5 — ScoreRing3D: segmented score torus with UnrealBloom and per-dimension reveal animation
Component
components/3d/ScoreRing3D.tsx
Geometry
TorusGeometry(1, 0.12, 32, 200) — high segment count for smooth arc edges
Segments
4 arcs: Relevance (teal) / Accuracy (cyan) / Clarity (violet) / Completeness (amber)
Animation
On mount: each arc draws itself via dashOffset 0→1 over 1.2s, staggered ease-out
Center Text
drei <Text3D> using Syne Bold; scales up with React Spring on load complete
Bloom
UnrealBloomPass postprocessing; bloom strength scales proportionally with score
Hover
Mouseover segment: segment scales to 1.1x, tooltip shows dimension label + score
Background
Starfield Points system with slow camera parallax on scroll
Dot Markers
White sphere placed at segment boundary joins for visual clarity
Props
scores: { relevance, accuracy, clarity, completeness }  — all 0–10 range
2.6 RoomParticles — Live Interview Ambient Scene
Fig 6 — RoomParticles3D: audio-reactive spiral particle system behind live interview room UI
Component
components/3d/RoomParticles.tsx
Geometry
Points (BufferGeometry) — 3000 particles in slow spiral formation
Motion
useFrame: particles orbit center; speed increases with audio amplitude
Color
Teal base with ±20° HSL hue variation per particle for organic variation
Audio React
Web Audio AnalyserNode connected to Stream video track; amplitude → speed
Canvas
Fixed full-screen, z-index -1, sits behind all UI elements
Performance
800 particles on mobile; useMemo geometry; DPR=1.0 on low-end devices
PostProcess
Subtle chromatic aberration pass via @react-three/postprocessing
Live UI
Stream Video tiles, Monaco code editor, Piston output render above canvas
Integration
Canvas positioned as background; React UI layers sit in front via z-index

4.2 Canvas3D Wrapper Pattern
All 3D components use this standard wrapper ensuring safe SSR, loading states, and graceful WebGL degradation:
// components/3d/Canvas3D.tsx
export default function Canvas3D({ children, height = 400 }) {
  return (
    <ErrorBoundary fallback={<FallbackUI />}>
      <Suspense fallback={<CanvasSkeleton height={height} />}>
        <Canvas
          dpr={[1, 1.5]}          // DPR cap for performance
          camera={{ fov:60, near:0.1, far:1000 }}
          style={{ height, background:'transparent' }}
          gl={{ antialias:true, alpha:true }}
        >
          <color attach='background' args={['transparent']} />
          {children}
        </Canvas>
      </Suspense>
    </ErrorBoundary>
  )
}
